#  Blinkit Sales & Customer Feedback Performance Dashboard

##  Project Overview

This project presents an interactive **Power BI dashboard** designed to analyze Blinkit’s business performance by combining **sales data, product insights, and customer feedback analysis** into a single reporting solution.

The dashboard enables stakeholders to track key metrics such as total orders, customer distribution, product performance, and sentiment trends, helping drive **data-driven decision-making**.

The dataset was sourced from a **CSV file**, transformed using **SQL**, and visualized using Power BI with interactive and dynamic components.

---

##  Key Objectives

* Analyze customer ordering behavior and segmentation
* Identify top-performing products and categories
* Evaluate regional sales performance
* Measure customer satisfaction through sentiment analysis
* Provide a centralized, interactive business dashboard

---

##  Tools & Technologies Used

* **Power BI Desktop** – Dashboard development
* **SQL** – Data cleaning, transformation, and loading
* **CSV Dataset** – Raw data source
* **DAX** – KPI calculations and measures

---

##  Data Workflow

2. Data cleaned and structured using SQL
3. Data loaded into Power BI
4. Data modeling and relationships created
5. DAX measures implemented
6. Dashboard designed with interactive visuals

##  Key Insights

* Certain cities (like Deoghar) dominate order volume
* Customer base is evenly distributed across segments
* Pet Care and Snacks categories drive the highest profits
* Majority of customers give mid-to-high ratings (3–4)
* Sentiment is evenly split, indicating scope for experience improvement
